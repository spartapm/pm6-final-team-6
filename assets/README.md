# Assets (source illustrations)

피그마/디자인에서 받은 원본 SVG입니다. 앱에서 쓰는 파일은 `public/illustrations/`에 복사·정리되어 있습니다.

| 소스 폴더 (요약) | public 경로 |
|------------------|-------------|
| 홈화면 상단 일러스트 | `home-hero.svg` |
| 홈화면 이번주 참여기록 이모지 (1~4) | `week-done-past.svg`, `week-done-today.svg`, `week-missed-past.svg`, `week-missed-today.svg` |
| 케어로그 기본 / 완료 일러스트 | `care-base-*.svg`, `care-done-*.svg` |
| 루틴 종료 · 난이도 이모지 | `end-hero.svg`, `end-diff-*.svg` |
| 변화 태그 선택 | `tags-hero-1.svg`, `tags-hero-2.svg`, `tags-hint.svg` |
| 마이페이지 기본 아바타 | `avatar-1.svg` ~ `avatar-4.svg` |
| 추천 데이터 없음 | `recommend-empty.svg` |
| 케어로그 변화 카드 | `change-card.svg` |

코드에서는 `@/lib/illustrations`의 `ILLUSTRATIONS` 상수를 사용합니다.
