1. 뒤로가기 버튼
- 클릭 시 이전 화면으로 이동한다.
- 입력 중인 정보가 있을 경우 저장되지 않을 수 있다는 확인 모달을 노출한다.

---------------------------------------------------------------------------------

2. 화면 타이틀 / 안내 문구
-정적 콘텐츠
- 입력한 피부 정보는 루틴 추천과 경험카드 탐색 기준으로 활용된다.

---------------------------------------------------------------------------------

3. 프로필 
-사진 
-닉네임 노출




---------------------------------------------------------------------------------

4. 피부 타입 선택
- 필수 항목이며 건성, 지성, 복합성, 민감성 중 1개만 선택 가능하다.
- 선택된 항목은 강조 상태로 표시된다.
- 선택값은 경험카드 추천, 피드 필터링, 유사 사용자 매칭 기준으로 활용된다.

---------------------------------------------------------------------------------

5. 피부 고민 선택
- 필수 항목이며 복수 선택 가능하다.
- 여드름/트러블
-색소침착
-모공/피지
-주름/탄력
-민감/붉음증
-보습/건조함
-미백/칙칙함
-각질
중에 선택한다.(왼쪽 5번 내에 단어가 아닌 해당 글 위에 있는 단어로 한다.)
- 최소 1개 이상 선택해야 한다.
- 선택값은 경험카드 추천 및 필터링 기준으로 활용된다.

---------------------------------------------------------------------------------

6. 피부 민감도 선택
- 필수 항목이며 낮음, 보통, 높음 중 1개만 선택 가능하다.
- 선택된 항목은 강조 상태로 표시된다.
- 선택값은 민감성 사용자 경험카드 추천과 루틴 추천 기준으로 활용된다.

---------------------------------------------------------------------------------

7. 저장하기 버튼
- 피부 타입, 피부 고민, 피부 민감도 선택이 완료되면 활성화된다.
- 클릭 시 피부 프로필 정보가 저장된다.
- 저장 후 루틴 등록 화면(3)으로 이동한다.
- 필수 항목이 누락된 경우 비활성화 상태를 유지하거나 누락 항목 안내 메시지를 노출한다.

---------------------------------------------------------------------------------
[저장하기 버튼 활성화 조건]
- 피부 타입 1개 선택
- 피부 고민 1개 이상 선택
- 피부 민감도 1개 선택
- 활성화 전 버튼 색 : F7EEEF
- 활성화 후 버튼 색 : FF617D
