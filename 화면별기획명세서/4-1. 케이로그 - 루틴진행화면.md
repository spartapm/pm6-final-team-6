4-1. 케어로그 - 루틴 진행 화면
1. 입주 구역 카드
화면 상단에 사용자가 입주한 피부 구역 카드를 노출한다.
카드 영역에는 유저가 선택한 피부 타입에 해당하는 캐릭터 이미지를 노출한다.
캐릭터 하단에는 현재 구역명을 표시한다.
구역명은 “(유저가 선택한 피부 타입) 구역 1등” 형식으로 표기한다.
카드 하단에는 “이제 이 피부 구역을 함께 기록해봐요.” 안내 문구를 노출한다.
입주 구역 카드는 조회 전용이며 별도 클릭 동작은 제공하지 않는다.
2. 오늘 했어요 버튼
화면 중앙에 “오늘 했어요” 버튼을 노출한다.
기본 상태에서는 비활성화 상태로 표시한다.
사용자가 오늘 루틴 단계 중 2개 이상을 완료 체크하면 “오늘 했어요” 버튼을 활성화한다.
활성화된 “오늘 했어요” 버튼 클릭 시 오늘의 케어로그 기록을 저장한다.
저장된 기록은 누적 기록일, 사용 기간, 스킨노트 생성 데이터에 반영한다.
저장 완료 후 “오늘의 케어로그가 저장되었어요.” 토스트 메시지를 노출한다.
이미 오늘 기록을 저장한 경우 버튼 문구를 “오늘 기록 완료”로 변경하고 비활성화한다.
이미 저장된 오늘 기록은 중복 저장되지 않는다.
3. 케어로그 헤더 / 루틴 선택하기 버튼
“케어로그” 제목을 카드 상단에 노출한다.
우측에는 “루틴 선택하기” 버튼을 노출한다.
등록된 루틴이 1개인 경우 현재 루틴을 기본으로 선택한다.
등록된 루틴이 2개 이상인 경우 “루틴 선택하기” 클릭 시 사용자가 등록한 루틴 목록을 드롭다운으로 노출한다.
등록된 루틴이 없는 경우 “진행 중인 루틴이 없어요.” 안내 문구와 “루틴 등록하기” CTA를 노출한다.
4. 케어로그 요약 카드
선택된 루틴의 오늘 진행 상태를 요약 카드로 노출한다.
요약 카드에는 다음 정보를 표시한다.
오늘 루틴 단계 수
완료한 단계 수
남은 단계 수
문구는 다음 형식으로 표시한다.
“오늘 루틴 4단계”
“완료 2개”
“남은 단계 2개”
사용자가 루틴 단계를 체크하거나 체크 해제할 경우 요약 카드의 완료 개수와 남은 단계 수를 즉시 업데이트한다.
5. 루틴 단계 체크리스트
선택된 루틴의 단계 목록을 사용 순서대로 노출한다.
각 단계는 카드 형태로 제공한다.
각 단계 카드에는 다음 정보를 표시한다.
단계 번호
제품 이미지
루틴 단계명
제품명
완료 체크박스
예시는 다음과 같은 형식으로 표시한다.
1 / 이미지 / 클렌징 / 약산성 클렌저
2 /  이미지 / 토너 / 진정 토너
3 /  이미지 / 세럼 / 시카 세럼
4 /  이미지 / 크림 / 장벽 크림
제품 이미지가 없는 경우 기본 이미지 플레이스홀더를 노출한다.
완료 체크박스 클릭 시 해당 단계의 완료 상태를 토글한다.
완료 체크된 단계는 “완료” 상태로 표시한다.
미완료 단계는 “미완료” 상태로 표시한다.
체크 상태가 변경될 때마다 케어로그 요약 카드의 완료 개수와 남은 단계 수를 업데이트한다.
체크 상태가 변경될 때마다 “오늘 했어요” 버튼 활성 조건을 다시 판단한다.
6. 단계별 완료 체크박스
각 루틴 단계 우측에 완료 체크박스를 노출한다.
체크박스 기본값은 미완료 상태이다.
체크박스 클릭 시 완료 상태로 변경된다.
완료 상태에서 다시 클릭하면 미완료 상태로 변경된다.
오늘 기록 저장 전까지는 사용자가 체크 상태를 수정할 수 있다.
“오늘 했어요” 버튼을 눌러 기록을 저장한 이후에는 당일 체크 상태를 수정할 수 없도록 한다.
수정이 필요한 경우 MVP 이후 기능으로 “오늘 기록 수정”을 제공할 수 있다.
7. 오늘 했어요 버튼 활성화 조건
등록된 루틴이 있어야 한다.
오늘 루틴 단계 목록이 노출되어 있어야 한다.
오늘 루틴 단계 중 2개 이상이 완료 체크되어야 한다.
위 조건을 모두 충족하면 “오늘 했어요” 버튼을 활성화한다.
조건을 충족하지 못할 경우 “오늘 했어요” 버튼은 비활성화 상태로 유지한다.
8. 캐릭터 이미지 변경 조건
기본 상태에서는 사용자의 피부 구역 또는 입주 상태를 나타내는 기본 캐릭터 이미지를 노출한다.
루틴 단계 중 일부만 완료한 경우 기본 캐릭터 이미지를 유지한다.
오늘의 루틴 전체 단계를 모두 완료하고 “오늘 했어요” 버튼을 클릭하면 캐릭터 이미지를 “전체 케어 완료” 상태 이미지로 변경한다.
변경된 이미지는 오늘 기록 저장 완료 상태를 시각적으로 알려주는 피드백으로 사용한다.
다음 날 새로운 케어로그가 시작되면 캐릭터 이미지는 다시 기본 상태로 초기화된다.
9. 하단 네비게이션 바
홈 클릭 시 홈 화면으로 이동한다.
케어로그 클릭 시 케어로그 화면으로 이동한다.
스킨서랍장 클릭 시 스킨서랍장 화면으로 이동한다.
마이페이지 클릭 시 마이페이지 화면으로 이동한다.
현재 화면은 케어로그이므로 케어로그 탭을 활성 상태로 표시한다.
현재 화면의 탭을 다시 클릭해도 별도 동작은 하지 않는다.
