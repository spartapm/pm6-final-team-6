9. 마이페이지 - 설정

1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
클릭 시 이전 화면(마이페이지)으로 이동한다.

2. 상단 타이틀 영역
화면 상단 중앙에 "설정" 텍스트를 노출한다.

3. 서비스 안내 영역
"서비스 안내" 그룹 타이틀을 노출한다.
하위 항목으로 공지사항, 문의하기, 이용약관을 리스트 형태로 노출한다.
각 항목 우측에 화살표(›) 아이콘을 노출한다.
공지사항 클릭 시 9-1 공지사항 화면으로 이동한다.
문의하기 클릭 시 9-2 문의하기 화면으로 이동한다.
이용약관 클릭 시 9-3 이용약관 화면으로 이동한다.

4. 계정 영역
"계정" 그룹 타이틀을 노출한다.
하위 항목으로 로그아웃, 회원탈퇴를 리스트 형태로 노출한다.
로그아웃 클릭 시 확인 모달("로그아웃 하시겠습니까?")을 노출하며, 확인 시 로그아웃 처리 후 비로그인 상태의 홈 화면으로 이동한다.
회원탈퇴 클릭 시 확인 모달("회원탈퇴 하시겠습니까?")을 노출하며, 확인 시 회원탈퇴 처리 후 비로그인 상태의 홈 화면으로 이동한다.
(탈퇴 시 데이터 처리 정책은 정책서 기준을 따른다)
모달 버튼: 아니요(좌·강조) / 예(우)
회원탈퇴 모달 안내: "탈퇴 시, 모든 정보가 삭제되며 복구되지 않습니다."

5. 하단 네비게이션 바
홈, 케어로그, 스킨서랍장, 마이페이지 탭을 노출한다.
마이페이지 탭이 활성 상태로 표시된다.
각 탭 클릭 시 해당 화면으로 이동한다.
