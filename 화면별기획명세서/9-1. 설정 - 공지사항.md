9-1. 설정 - 공지사항

1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
클릭 시 설정 화면으로 이동한다.

2. 상단 타이틀 영역
화면 상단 중앙에 "공지사항" 텍스트를 노출한다.

3. 공지글 리스트 영역
등록된 공지글을 카드 형태로 리스트업하여 노출한다.
최신 등록순으로 정렬하여 노출한다.
각 카드에는 공지글 제목을 노출한다.
제목이 화면 너비를 초과할 경우 말줄임(…) 처리한다.
카드 클릭 시 9-1-1(설정 - 공지사항 - 게시글) 화면으로 이동하며, 해당 공지글의 상세 내용이 노출된다.
등록된 공지글이 없는 경우 "등록된 공지사항이 없어요" 빈 상태 문구를 노출한다.

4. 하단 네비게이션 바
홈, 케어로그, 스킨서랍장, 마이페이지 탭을 노출한다.
마이페이지 탭이 활성 상태로 표시된다.
각 탭 클릭 시 해당 화면으로 이동한다.
