루틴 종류 버튼 
클릭하면 테두리 색 ‘강조’ 색상으로 변경되며 선택됨. 
한가지만 선택 가능. 다른 버튼 클릭 시 기존에 선택되었던 버튼은 원래 색상으로 변경되며 선택해제됨.

제품 검색 탭 
상단 검색바에 등록하려는 제품을 검색하면 관련도 높은 제품이 하단에 리스트형식으로 표기됨. 
검색 결과는 좌측에서 선택한 루틴 단계(카테고리)에 맞는 제품만 자동 필터링되어 노출된다. (온오프 옵션 없음)
네이버 쇼핑 category2/3/4·상품명 키워드와 단계명을 매칭하며, 카테고리 필터 후에도 긴 제품명은 말줄임 대신 슬라이드(좌우 이동)로 전체 이름을 확인할 수 있다.
여러 상품이 노출될 시, 스크롤 형태로 표기되어 아래로 스크롤 시 아래 제품 노출됨. 
제품 클릭 시, 오른쪽 체크박스에 체크표시 되며 선택됨. 재클릭 시 선택해제됨
선택시 해당 테두리 색 asdf(예시) 색상으로 표기.
