8-3-2. 비밀번호 찾기 - 인증번호 확인
1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 비밀번호 찾기 1단계 화면으로 이동한다.
2. 상단 타이틀 영역
화면 상단 중앙에 “비밀번호 찾기” 텍스트를 노출한다.
타이틀 하단에는 “2 / 3 · 이메일로 전송된 인증번호를 입력해주세요” 안내 문구를 노출한다.
3. 인증번호 입력란
인증번호 입력란을 노출한다.
입력란 placeholder는 “인증번호 6자리 입력”으로 표시한다.
인증번호는 숫자 6자리만 입력 가능하다.
인증번호 입력란 우측에는 남은 인증 시간을 “03:00” 형식으로 노출한다.
인증번호 6자리가 입력되면 “확인” 버튼을 활성화한다.
인증번호가 6자리 미만인 경우 확인 버튼은 비활성화 상태로 표시한다.
4. 인증번호 타이머
인증번호 발송 성공 시 3분 타이머를 시작한다.
타이머가 종료되면 입력된 인증번호를 비활성화하고, 입력창 하단에 “인증번호가 만료되었습니다. 다시 시도해주세요.” 안내 문구를 노출한다.
인증번호 재전송 시 타이머를 3분으로 초기화한다.
5. 인증번호 재전송
인증번호 입력란 하단에 “인증번호 재전송” 텍스트 버튼을 노출한다.
인증번호 재전송 클릭 시 동일한 이메일로 인증번호를 다시 발송한다.
재전송 성공 시 입력된 인증번호를 초기화하고 타이머를 3분으로 재시작한다.
재전송 실패 시 “인증번호 재전송에 실패했습니다. 잠시 후 다시 시도해주세요.” 안내 문구를 노출한다.
6. 확인 버튼
화면에 “확인” 버튼을 노출한다.
인증번호 6자리가 입력되기 전까지 버튼은 비활성화 상태로 표시한다.
확인 버튼 클릭 시 인증번호 검증을 요청한다.
인증번호가 일치하는 경우 비밀번호 찾기 3단계 화면으로 이동한다.
인증번호가 일치하지 않는 경우 입력창 하단에 “인증번호가 일치하지 않습니다.” 안내 문구를 노출한다.
인증번호가 만료된 경우 입력창 하단에 “인증번호가 만료되었습니다. 다시 시도해주세요.” 안내 문구를 노출한다.
7. 하단 진행 안내 문구
화면 하단에 “화면 2/3 — 인증번호 확인” 문구를 노출한다.
