5. 루틴 종료 화면
1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 이전 화면으로 이동한다.
입력 중인 종료 정보가 저장되지 않은 경우 “작성 중인 내용이 저장되지 않았어요. 나가시겠어요?” 확인 모달을 노출한다.
확인 모달에서 “나가기” 선택 시 이전 화면으로 이동한다.
“계속 작성” 선택 시 모달을 닫고 현재 화면에 머무른다.
2. 상단 타이틀 영역
화면 상단에 루틴 종료 캐릭터 이미지를 노출한다.
캐릭터 우측 또는 하단에 “이번 루틴, 어떠셨나요?” 타이틀을 노출한다.
화면 제목은 “루틴 종료”로 표기한다.
3. 종료 사유 선택 영역
“종료 사유” 영역을 노출한다.
안내 문구는 “해당하는 이유를 선택해 주세요.”로 표기한다.
종료 사유는 필수 선택 항목이다.
종료 사유는 1개만 선택할 수 있다.
선택지는 다음 3개로 제공한다.
변화가 느껴져서 마칠래요
변화는 없지만 기록을 마칠래요
지속하기 어려워서 그만할래요
“지속하기 어려워서 그만할래요” 선택 시 세부 사유 팝업을 노출한다.
팝업 제목: “지속하기 어려웠던 이유가 무엇인가요?”
팝업 안내: “서비스 개선을 위해 가장 가까운 이유를 선택해주세요. (최대 2개까지)”
세부 사유는 다음 6개이며 복수 선택 가능(최대 2개), “기타”/자유 서술은 제공하지 않는다.
매일 기록하는 게 번거로웠어요 / 루틴 단계가 너무 많았어요 / 사용하는 제품이 바뀌었어요 / 피부에 자극이 있었어요 / 효과를 느끼지 못했어요 / 루틴을 자주 잊어버렸어요
2개 선택 시 나머지 항목은 비활성 처리한다. “선택 완료”는 1개 이상 선택 시 활성화된다. X 클릭 시 팝업을 닫고 루틴 종료 화면으로 복귀한다.
선택된 항목은 활성 상태로 표시한다.
미선택 상태에서 “이번 루틴 마치기” 버튼을 누를 수 없도록 비활성화한다.
4. 루틴 난이도 선택 영역
“이번 루틴은 꾸준히 하기 어땠나요?” 질문을 노출한다.
루틴 난이도는 필수 선택 항목이다.
난이도는 1개만 선택할 수 있다.
선택지는 다음 3개로 제공한다.
쉬웠어요
보통이에요
어려웠어요
선택된 항목은 활성 상태로 표시한다.
미선택 상태에서 “이번 루틴 마치기” 버튼을 누를 수 없도록 비활성화한다.
5. 변화 태그 선택 영역
“어떤 변화가 있었나요?” 질문을 노출한다.
변화 태그는 필수 항목이다.
기본 상태에서는 대표 변화 태그 일부를 노출한다.
태그는 복수 선택 가능하다.
사용자는 최대 5개까지 선택할 수 있다.
선택된 태그는 활성 상태로 표시한다.
우측 화살표 버튼 클릭 시 변화 태그 전체 선택 화면으로 이동한다.
6. 체감 변화 선택 영역
“체감 변화” 영역을 노출한다.
안내 문구는 “이번 루틴을 통해 느낀 변화를 선택해 주세요.”로 표기한다.
체감 변화는 선택 항목이다.
별점은 1점 단위로 선택 가능하다.
선택 범위는 0점부터 5점까지 제공한다.
예시는 다음과 같다.
0점: 전혀 없어요
5점: 매우 많이 느껴요
선택된 별점은 활성 상태로 표시한다.
미선택 상태에서도 종료 사유와 루틴 난이도가 선택되어 있으면 루틴 종료가 가능하다.
7. 이번 루틴 마치기 버튼
화면 하단에 “이번 루틴 마치기” 버튼을 노출한다.
기본 상태에서는 비활성화 상태로 표시한다.
종료 사유 1개와 루틴 난이도 1개, 변화태그가 모두 선택되면 버튼을 활성화한다.
체감 변화는 선택 항목이므로 버튼 활성 조건에 포함하지 않는다.
버튼 클릭 시 루틴 종료 정보를 저장한다.
저장되는 정보는 다음과 같다.
종료 사유
루틴 난이도
변화 태그
체감 변화 별점
루틴 ID
종료 날짜
저장 완료 후 스킨노트 생성 화면으로 이동한다.
저장 실패 시 “루틴 종료에 실패했어요. 다시 시도해주세요.” 안내 문구를 노출한다.
저장 중에는 버튼을 로딩 상태로 표시하고 중복 클릭을 방지한다.
