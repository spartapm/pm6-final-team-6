9-3. 설정 - 이용약관

1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
클릭 시 이전 화면(설정)으로 이동한다.

2. 상단 타이틀 영역
화면 상단 중앙에 "이용약관" 텍스트를 고정 노출한다.
타이틀은 선택된 탭과 무관하게 "이용약관"으로 동일하게 유지된다.

3. 탭 메뉴 영역
"이용약관", "개인정보 처리방침" 2개 탭을 가로로 노출한다.
진입 시 기본 탭은 "이용약관"으로 선택된 상태로 노출한다.
선택된 탭은 활성 스타일(밑줄과 강조 색상)로 표시되고, 미선택 탭은 비활성 스타일로 표시된다.
탭 클릭 시 다른 탭으로 전환되며, 새로 선택된 탭만 활성 상태로 변경된다.
"이용약관" 탭 클릭 시 이용약관 본문을 노출한다.
"개인정보 처리방침" 탭 클릭 시 개인정보 처리방침 본문을 노출한다.
탭 전환 시 스크롤 위치는 각 탭 진입 시 최상단으로 초기화된다.
탭 전환은 별도 화면 이동 없이 동일 화면 내에서 콘텐츠만 교체되는 방식으로 동작한다.

4. 본문 콘텐츠 영역
선택된 탭에 해당하는 약관 본문을 텍스트로 노출한다.
본문은 장(예: 제1장 총칙) - 조(예: 제1조 목적) - 항(숫자) - 호(하위 숫자) 구조로 노출된다.
조 제목은 굵게(강조) 표시하고, 본문 텍스트와 시각적으로 구분한다.
본문 내용은 조회 전용이며, 텍스트 클릭 시 별도 동작이 발생하지 않는다.
본문 내용이 화면 높이를 초과할 경우 세로 스크롤로 열람 가능하다.
링크, 이메일 주소 등 별도 텍스트가 포함되어도 클릭 가능한 링크로 동작하지 않는다.

5. 하단 네비게이션 바
홈, 케어로그, 스킨서랍장, 마이페이지 탭을 노출한다.
마이페이지 탭이 활성 상태로 표시된다.
각 탭 클릭 시 해당 화면으로 이동한다.
