1. 상단 인사 / 메인 메시지 영역
메인 메시지 문구 (와프 표기)
일러스트 영역: 이미지 추가(첨부) 

------------------------------------------------------------------------------

2. 내 피부 프로필 영역
피부 프로필 카드 영역은 블러 처리
버튼 형식
안내 문구: (와프 표기)
버튼 클릭 →  8-1. (마이페이지 - 비로그인일 경우 로그인 페이지) 이동

------------------------------------------------------------------------------

3. 명예의 스킨노트 영역
현재 사용자는 스킨노트 미공유 상태이므로 카드 상세 내용은 블러 처리  
더보기 클릭 → 7번(경험게시판)이동

------------------------------------------------------------------------------

4. 이번 주 참여 기록 빈 상태 영역
루틴 미등록 상태이므로 실제 주간 기록 데이터 없음
빈 상태 카드 문구: 와프 표기
말풍선 문구: 와프 표기 → 내일 이야기 해보기 
영역 클릭→ 해당 영역은 정보 확인용 영역으로 클릭 동작 없음

------------------------------------------------------------------------------

5. 하단 네비게이션 바
현재 선택된 탭 이모지 강조
[홈], [케어 로그], [경험 게시판], [마이페이지] 구성
[홈] → 1번 화면(홈 화면) 이동
[케어로그] → 8-1. (마이페이지 - 비로그인일 경우 로그인 페이지) 이동
[경험 게시판] → 7번 화면(스킨 서랍장) 이동
[마이페이지] → 8-1. (마이페이지 - 비로그인일 경우 로그인 페이지) 이동

------------------------------------------------------------------------------


스킨노트 열람 기준
스킨노트 공유 없음: 경험 게시물 1일 1개 열람 가능
스킨노트 공유(사진 없음, 주차별 태그 있음): 경험 게시물 3일에 10개 열람 가능
스킨노트 공유(사진 포함): 경험 게시물 전체 열람 가능
- 1일 열람 가능 횟수 초과 시 열람 제한 안내 모달 노출
