5-1. 변화 태그 선택 화면
1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 이전 화면인 루틴 종료 화면으로 이동한다.
태그를 선택하거나 변경한 상태에서 뒤로가기 버튼을 클릭하면 “선택한 태그가 저장되지 않았어요. 나가시겠어요?” 확인 모달을 노출한다.
확인 모달에서 “나가기”를 선택하면 선택값을 저장하지 않고 루틴 종료 화면으로 이동한다.
“계속 선택”을 선택하면 모달을 닫고 현재 화면에 머무른다.
태그 선택 또는 변경 내역이 없는 경우 확인 모달 없이 이전 화면으로 이동한다.
2. 상단 타이틀 영역
화면 상단 중앙에 “변화 태그 선택” 텍스트를 노출한다.
타이틀 하단에는 “루틴 사용 후 변화를 선택해주세요” 안내 문구를 노출한다.
타이틀 하단에 캐릭터 이미지를 노출한다.
캐릭터 이미지는 조회 전용이며 별도 클릭 동작은 제공하지 않는다.
3. 변화 태그 선택 영역
“이번 루틴에서 느낀 변화를 선택해주세요” 안내 문구를 노출한다.
현재 선택한 태그 개수와 최대 선택 가능 개수를 “3/5” 형식으로 표시한다.
우측에는 “최대 5개” 안내 배지를 노출한다.
태그는 버튼 형태로 제공한다.
사용자는 태그를 최소 1개 이상, 최대 5개까지 선택할 수 있다.
선택된 태그는 활성 상태로 표시한다.
선택 해제 시 활성 상태와 체크 표시를 제거한다.
최대 5개를 선택한 상태에서 추가 태그를 선택할 경우 “태그는 최대 5개까지 선택할 수 있어요.” 안내 문구를 노출한다.
4. 태그 선택 규칙
일반 변화 태그는 복수 선택할 수 있다.
“#큰 변화 없음”은 다른 태그와 동시에 선택할 수 없는 단독 옵션으로 처리한다.
다른 태그가 1개 이상 선택된 상태에서 “#큰 변화 없음”을 선택하면 기존 선택 태그를 모두 해제하고 “#큰 변화 없음”만 선택한다.
“#큰 변화 없음”이 선택된 상태에서 다른 태그를 선택하면 “#큰 변화 없음” 선택을 해제하고 새로 선택한 태그만 선택 상태로 처리한다.
“#큰 변화 없음”을 선택한 경우 선택 개수는 1개로 계산한다.
5. 선택 완료 버튼
화면 하단에 “선택 완료” 버튼을 노출한다.
태그를 1개 이상 선택하기 전까지 버튼은 비활성화 상태로 표시한다.
태그를 1개 이상 선택하면 버튼을 활성화한다.
선택 개수에 따라 버튼 문구를 “선택 완료” 또는 “선택 완료 (3)” 형태로 표시할 수 있다.
선택 완료 버튼 클릭 시 선택한 태그 값을 루틴 종료 화면으로 전달한다.
선택 완료 후 루틴 종료 화면으로 이동한다.
루틴 종료 화면의 변화 태그 선택 영역에 선택한 태그를 반영한다.
최종 저장은 루틴 종료 화면의 “이번 루틴 마치기” 버튼 클릭 시 처리한다.
6. 안내 카드 영역
화면 하단에 안내 카드 영역을 노출한다.
안내 문구는 “태그는 스킨노트 생성과 루틴 분석에 활용돼요.”로 표시한다.
안내 카드 영역은 조회 전용이며 별도 클릭 동작은 제공하지 않는다.
