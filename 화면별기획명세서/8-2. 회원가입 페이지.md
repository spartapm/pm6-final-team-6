1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 로그인 화면으로 이동한다.
2. 상단 타이틀 영역
화면 상단 중앙에 “회원가입” 텍스트를 노출한다.
타이틀 하단에는 “내 피부 고민에 맞는 루틴을 기록해요” 안내 문구를 노출한다.
3. 닉네임 입력란
닉네임 입력란을 노출한다.
입력란 placeholder는 “닉네임 입력”으로 표시한다.
닉네임은 필수 입력 항목이다.
닉네임 미입력 상태에서 회원가입을 시도할 경우 입력창 하단에 “닉네임을 입력해주세요.” 안내 문구를 노출한다.
이미 사용 중인 닉네임인 경우 입력창 하단에 “이미 사용 중인 닉네임입니다.” 안내 문구를 노출한다.
4. 이메일 입력란
이메일 입력란을 노출한다.
입력란 placeholder는 “이메일 입력”으로 표시한다.
이메일은 필수 입력 항목이다.
이메일 미입력 상태에서 회원가입을 시도할 경우 입력창 하단에 “이메일을 입력해주세요.” 안내 문구를 노출한다.
이메일 형식이 올바르지 않은 경우 입력창 하단에 “올바른 이메일 형식으로 입력해주세요.” 안내 문구를 노출한다.
이미 가입된 이메일인 경우 입력창 하단에 “이미 가입된 이메일입니다.” 안내 문구를 노출한다.
5. 비밀번호 입력란
비밀번호 입력란을 노출한다.
입력란 placeholder는 “비밀번호 입력 (8자 이상, 영문+숫자)”로 표시한다.
비밀번호는 필수 입력 항목이다.
비밀번호는 8자 이상, 영문과 숫자를 포함해야 한다.
비밀번호는 마스킹 처리하여 표시한다.
비밀번호 미입력 상태에서 회원가입을 시도할 경우 입력창 하단에 “비밀번호를 입력해주세요.” 안내 문구를 노출한다.
조건을 충족하지 않을 경우 입력창 하단에 “비밀번호는 8자 이상, 영문과 숫자를 포함해야 합니다.” 안내 문구를 노출한다.
6. 비밀번호 확인 입력란
비밀번호 확인 입력란을 노출한다.
입력란 placeholder는 “비밀번호 확인”으로 표시한다.
비밀번호 확인은 필수 입력 항목이다.
비밀번호 확인 값은 마스킹 처리하여 표시한다.
비밀번호와 일치하지 않을 경우 입력창 하단에 “비밀번호가 일치하지 않습니다.” 안내 문구를 노출한다.
7. 연령대 선택
연령대 선택 드롭다운을 노출한다.
선택 전 기본 placeholder는 “연령대 선택”으로 표시한다.
옵션은 10대 / 20대 / 30대 / 40대 / 50대 이상으로 제공한다.
연령대는 필수 선택 항목이다.
미선택 상태에서 회원가입을 시도할 경우 “연령대를 선택해주세요.” 안내 문구를 노출한다.
8. 성별 선택
성별 선택 버튼을 노출한다.
옵션은 여성 / 남성으로 제공한다.
선택된 성별 버튼은 활성 상태로 표시한다.
성별은 선택 항목으로 제공한다.
성별을 선택하지 않아도 회원가입은 가능하다.
9. 이용약관 동의
“[필수] 이용약관 동의” 체크박스를 노출한다.
체크박스 클릭 시 선택/해제 상태가 토글된다.
“보기” 클릭 시 이용약관 상세 화면 또는 모달을 노출한다.
미동의 상태에서 회원가입을 시도할 경우 “이용약관에 동의해주세요.” 안내 문구를 노출한다.
10. 개인정보 처리방침 동의
“[필수] 개인정보 처리방침 동의” 체크박스를 노출한다.
체크박스 클릭 시 선택/해제 상태가 토글된다.
“보기” 클릭 시 개인정보 처리방침 상세 화면 또는 모달을 노출한다.
미동의 상태에서 회원가입을 시도할 경우 “개인정보 처리방침에 동의해주세요.” 안내 문구를 노출한다.
11. 회원가입 버튼
화면 하단에 “회원가입” 버튼을 노출한다.
필수 입력값과 필수 약관 동의가 완료되기 전까지 버튼은 비활성화 상태로 표시한다.
필수 입력값과 필수 약관 동의가 완료되면 버튼을 활성화한다.
회원가입 버튼 클릭 시 회원가입을 요청한다.
회원가입 성공 시 피부 프로필 등록 화면으로 이동한다.
회원가입 실패 시 실패 원인에 맞는 안내 문구를 노출한다.
12. 로그인 이동 링크
회원가입 버튼 하단에 “이미 계정이 있으신가요? 로그인” 문구를 노출한다.
“로그인” 텍스트 클릭 시 로그인 화면으로 이동한다.
