기능설명(코치마크)

공통
- 화면 우측 상단 `?` / "기능설명" 아이콘 클릭 시에만 해당 화면 설명 플로우 실행
- 최초 1회 자동 온보딩 튜토리얼과 별개 (언제든 재확인 가능)
- 다음/이전/건너뛰기 버튼 없음 — 하이라이트 외 빈 영역 클릭 시 다음 단계
- 이전으로 되돌아가기 없음
- 하이라이트 대상 클릭 시 원본 인터랙션·다음 이동 모두 없음
- 하단 점 인디케이터(●○○○○)로 진행 표시 (클릭 불가)
- 마지막 단계 빈 영역 클릭 시 종료 후 원래 화면 유지

홈(home) 순서
1. 내 피부 프로필 — 피부 타입과 고민이 아직 비어 있으면 얼른 등록해요
2. 오늘 루틴 기록하기 — 오늘 한 스킨케어를 기록해가는 가장 중요한 시작 버튼이에요
3. 명예의 스킨노트 — 저장 best, 댓글 best 기록을 빠르게 볼 수 있어요
4. 이번주 참여 기록 — 아직 루틴을 시작하지 않았는지, 이번 주 기록이 있는지 한눈에 확인해요
5. 하단 탭 이동 — 홈은 시작점, 케어로그는 기록, 스킨서랍장은 저장한 후기, 마이페이지는 내 정보예요

피부 프로필 등록(skin-profile) 순서
1. 프로필 확인 — 내 이름과 프로필을 먼저 확인해요
2. 피부 타입 선택 — 건성·지성·복합성·민감성 중 내 피부에 가장 가까운 타입을 골라요
3. 피부 고민 선택 — 여드름/트러블, 색소침착, 민감/붉음증처럼 내 고민을 선택해요
4. 민감도 선택 — 낮음·보통·높음 중 피부 자극 민감도를 체크해요
5. 저장하기 — 필수 항목을 선택하면 저장 버튼이 활성화돼요

루틴 등록(routine-register) 순서 — 직접 만들기 기준
1. 등록 방식 선택 — 직접 루틴 만들기 / 피부 고민 맞춤 추천 탭
2. 루틴 순서 설정 — 단계 삭제(−) · 드래그로 순서 변경
3. 단계별 제품 추가 — 단계 추가하기 버튼
4. 제품 검색 — 검색창 (시트 자동 오픈)
5. 직접 제품 등록하기 — 링크 (시트 유지)
6. 루틴 설정 완료 — 완료 버튼

루틴 등록 맞춤 추천(routine-register-recommend) 순서
1. 등록 방식 선택 — 지금은 '맞춤 추천'을 알려드릴게요
2. 이렇게 추천했어요 — 피부 타입/고민/민감도 태그
3. 추천 루틴 미리보기 — 제품 가로 스크롤 카드
4. 추천 루틴 적용 / 다른 루틴 추천받기
5. 루틴 설정 완료 — 적용 후 활성화

※ 기능설명 버튼은 현재 선택 탭에 맞는 투어를 실행한다.

케어로그(care-log) 순서
1. 루틴 진행 현황 — 시작일 · 경과일
2. 오늘 단계 체크 — 단계별 체크리스트
3. 완료/남은 단계 보기 — 요약 바
4. 오늘 했어요 — 기록 완료 버튼
5. 변화 과정 기록 상태 — 주간 기록 배지
6. 루틴 종료 — 상단 종료 아이콘

변화 과정 기록(care-log-change) 순서
1. 변화 사진 — 사진 업로드 슬롯 (선택)
2. 변화 여부 선택 — 있었어요/모르겠어요/없었어요
3. 변화 태그 선택 — 태그 목록
4. 복수 선택 가능 — 안내 문구
5. 기록 저장하기 — 저장 버튼

루틴 종료(routine-end) 순서
1. 종료 사유 선택
2. 꾸준히 하기 어땠는지 평가 — 이모지
3. 어떤 변화가 있었는지 선택 — 태그 진입 영역
4. 체감 변화 — 별점
5. 이번 루틴 마치기

변화 태그 선택(routine-end-tags) 순서
1. 변화 태그 선택 — 일반 태그 목록
2. 큰 변화 없음 — 단독 선택 태그
3. 선택 완료

스킨노트 완성(skin-note-complete) 순서
1. 스킨노트 저장하기
2. 피부 타입/고민 요약
3. 사용 제품 & 사용 기간 / 루틴 순서
4. 루틴 난이도 & 변화 과정
5. 변화 태그 & 체감 변화 & 루틴 종료 사유
6. 나만 보기 / 공유하고 스킨서랍장 구경하기

스킨 서랍장(drawer) 순서
1. 조건 선택 — 드롭다운 + 필터 칩
2. 사진 토글
3. 후기 카드 제목
4. 핵심 정보 읽기 — 사용 기간 / 체감 변화
5. 반응 수치 보기 — 저장 / 도움돼요 / 댓글

스킨노트 상세(note-detail) 순서
1. 제품/루틴 보기 — 사용 제품 썸네일
2. 변화 과정 보기 — 일차별 사진 + 사용 기간/체감 변화
3. 저장하기
4. 도움돼요 / 댓글
5. 댓글 입력

구현

다른 화면(케어로그·서랍장·상세 등) 순서별 문구가 확정되면 tours.ts에 동일 패턴으로 추가한다.
