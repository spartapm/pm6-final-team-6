경험카드 저장하기 버튼
-클릭 시 경험카드가 이미지 파일로 사용자 갤러리에 저장된다.
--------------------------------------------------------------------

피부타입, 피부 고민
- 작성자의 피부 프로필 정보에 따른 데이터를 표기한다.
--------------------------------------------------------------------

3. 사용 제품 / 사용 기간 / 루틴 순서 
- 해당 루틴에 등록되어 있는 제품의 이미지를 표기한다.  
- 사용 기간은 루틴 지속 기간을 표기한다. 
- 루틴 순서는 해당 루틴에 등록되어 있는 '제품 카테고리 종류'를 표기한다.
--------------------------------------------------------------------

4. 루틴 난이도 
- 5번(루틴 종료) 화면에 있는 '루틴 난이도 선택 영역'에서 선택된 데이터를 표기한다.

--------------------------------------------------------------------

5. 변화 과정 
 [사진 등록 유저]
  등록했던 사진 노출.  초과 시 오른쪽으로 스크롤하여 열람 
 [사진 미등록 유저] 
  4-2(변화 과정 기록)화면에서 선택한 '피부 변화 여부’ 이모지 데이터 노출
  초과 시 오른쪽으로 스크롤하여 열람

--------------------------------------------------------------------

6. 변화 태그 / 체감 변화 / 루틴 종료 사유 
- 변화 태그 5-1(변화 태그 선택 영역)에서 선택한 데이터를 표기
   가로줄에 최대 3개까지 표기하고 초과 시 줄바꿈 
- 선택 범위 : 1점부터 ~ 5점
- 체감 변화는 5번 화면의 '체감 변화 선택 영역'에서 선택한 데이터를 표기 
- 루틴 종료 사유는 5번(루틴 종료' 화면)에서 선택한 종료 사유 데이터를 표기
--------------------------------------------------------------------

7. 나만 보기 버튼 
- 클릭 시 해당 스킨노트가 서버에 저장된다. 
- 저장된 스킨노트는 로그인한 사용자 계정에 귀속되며, 이후 재로그인 시에도 마이페이지 내 '스킨노트 모아보기’에서 조회 
- 클릭 시 → 8번화면( 마이페이지)으로 이동

--------------------------------------------------------------------

8. 공유하고 스킨서랍장 구경하기 버튼
- 클릭 시 해당 스킨노트가 '스킨서랍장'에 게시된다. 
- 게시 완료 시 7번 화면으로 이동한다.
--------------------------------------------------------------------

9. 경험카드 생성날짜 
- 해당 경험카드를 생성시킨 루틴 등록 날짜를 표기한다.
