4-2. 변화 과정 기록 화면

1. 진입 조건
케어로그 화면의 “이번주 변화과정기록” 카드를 클릭하면 변화 과정 기록 화면으로 이동한다.
변화 과정 기록은 주 1회 작성할 수 있다.   

2. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
입력한 내용이 없는 상태에서 뒤로가기 버튼을 클릭하면 케어로그 화면으로 이동한다.
입력한 내용이 있는 상태에서 뒤로가기 버튼을 클릭하면 “작성 중인 기록을 저장하지 않고 나갈까요?” 확인 모달을 노출한다.
확인 모달의 “나가기” 클릭 시 입력 내용을 저장하지 않고 케어로그 화면으로 이동한다.
확인 모달의 “계속 작성” 클릭 시 모달을 닫고 현재 화면에 머무른다.
  
3. 상단 타이틀 영역
화면 상단에 “변화 과정 기록” 타이틀을 노출한다.
타이틀 하단에는 “피부 변화 과정을 기록해보세요.” 안내 문구를 노출한다.
추가 안내 문구로 “기록할 수록 루틴이 더 정확해져요.”를 노출한다.

4. 이번 주 변화 사진 영역
“이번 주 변화 사진” 제목을 노출한다.
사진 등록은 선택사항이다.
주 1회 사진을 등록 할 수 있으며, 주차별로 사진이 업로드 된다.
빈 사진 슬롯에는 “사진 추가” 아이콘과 텍스트를 노출한다.
빈 슬롯 클릭 시 카메라 촬영 또는 앨범 선택 옵션을 제공한다.
등록 가능한 파일 형식은 jpg, png로 제한한다.
파일 용량은 1장당 최대 10MB로 제한한다.
조건을 충족하지 않는 파일을 선택할 경우 “jpg, png 파일만 등록할 수 있어요.” 또는 “사진은 10MB 이하로 등록해주세요.” 안내 문구를 노출한다.
등록된 사진 우측 상단에는 삭제 아이콘을 노출한다.
삭제 아이콘 클릭 시 해당 사진을 삭제하고 빈 슬롯 상태로 되돌린다.
 
5. 사진 미등록 안내 영역
사진을 등록하지 않은 사용자도 변화 과정을 기록할 수 있도록 안내 문구를 노출한다.
안내 문구는 “사진을 등록하지 않아도 기록할 수 있어요. 아래에서 변화 여부와 태그를 선택해주세요.”로 표시한다.
해당 영역은 조회 전용이며 별도 클릭 동작은 제공하지 않는다.

6. 이번 주 피부 변화 선택
“이번 주 피부 변화는 어땠나요?” 질문을 노출한다.
선택지는 이모지 버튼 형태로 제공한다.
선택지는 다음 3개로 제공한다.
변화가 있었어요
모르겠어요
변화가 없었어요
사용자는 1개만 선택할 수 있다.
선택된 항목은 활성 상태로 표시한다.
기본값은 미선택 상태이다.
변화 여부는 필수 선택 항목이다.

7. 변화 태그 선택
“어떤 변화가 있었나요?” 질문을 노출한다.
변화 태그는 복수 선택 가능하다.
사용자는 최소 1개 이상, 최대 5개까지 선택할 수 있다.
선택된 태그는 활성 상태로 표시한다.
선택 가능한 태그 예시는 프레임 제목: 태그 리스트

8. 기록 저장하기 버튼
화면 하단에 “기록 저장하기” 버튼을 노출한다.
변화 여부가 선택되지 않은 경우 버튼은 비활성화 상태로 표시한다.
변화 여부를 선택하고 변화 태그를 1개 이상 선택하면 버튼을 활성화한다.
사진 등록 여부와 관계없이 변화 여부와 변화 태그가 선택되면 기록 저장이 가능하다.
버튼 클릭 시 이번 주 변화 과정 기록을 저장한다.
저장되는 정보는 다음과 같다.
등록 사진
변화 여부 
변화 태그
기록 날짜
루틴 ID
저장 완료 후 “이번 주 변화 과정이 저장되었어요.” 토스트 메시지를 노출한다.
저장 완료 후 4번화면(케어로그)으로 이동한다.
저장된 변화 과정 기록은 루틴 종료 후 스킨노트 생성 데이터에 반영된다.

9. 예외 상태
사진 업로드 실패 시 “사진 업로드에 실패했어요. 다시 시도해주세요.” 안내 문구를 노출한다.
네트워크 오류로 기록 저장에 실패한 경우 “기록 저장에 실패했어요. 잠시 후 다시 시도해주세요.” 안내 문구를 노출한다.
저장 중에는 “기록 저장하기” 버튼을 로딩 상태로 표시하고 중복 클릭을 방지한다.
