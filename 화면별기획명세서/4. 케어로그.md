4. 케어로그 화면
1. 입주 구역 카드
화면 상단에 사용자가 선택한 피부 고민에 맞는 구역 카드를 노출한다.
카드 영역에는 유저가 선택한 피부 타입 캐릭터 이미지를 노출한다.
캐릭터 하단에는 현재 입주 구역명을 표시한다.
구역명 예시는 “(피부고민) 구역”으로 표기한다.
카드 하단에는 “이제 이 피부 구역을 함께 기록해봐요.” 안내 문구를 노출한다.
입주 구역 카드는 조회 전용이며 별도 클릭 동작은 제공하지 않는다.
2. 오늘 했어요 버튼
화면 중앙에 “오늘 했어요” 버튼을 노출한다.
기본 상태에서는 비활성화 상태로 표시한다.
사용자가 “오늘 루틴 보기”를 펼친 뒤, 오늘 루틴 단계 중 2개 이상을 체크하면 “오늘 했어요” 버튼을 활성화한다.
활성화된 “오늘 했어요” 버튼 클릭 시 오늘 루틴 기록을 저장한다.
저장된 기록은 누적 기록일, 사용 기간, 경험카드 생성 데이터에 반영한다.
기록 저장 완료 후 “오늘의 케어로그가 저장되었어요.” 토스트 메시지를 노출한다.
이미 오늘 기록을 완료한 경우 버튼 문구는 “오늘 기록 완료”로 변경하고 비활성화한다.
3. 케어로그 카드 / 오늘 루틴 보기
“케어로그” 카드를 노출한다.
카드 우측에는 “오늘 루틴 보기” 버튼을 노출한다.
기본 상태에서는 루틴 상세 목록을 접힌 상태로 표시한다.
“오늘 루틴 보기” 버튼 클릭 시 오늘 루틴의 상세 단계 목록을 펼쳐서 노출한다.
펼쳐진 목록에는 루틴 단계, 사용 제품명, 체크박스를 표시한다.
사용자는 오늘 수행한 루틴 단계를 체크할 수 있다.
다시 “오늘 루틴 보기” 버튼을 클릭하면 상세 목록을 접는다.
루틴 상세 목록이 접힌 상태에서는 “루틴 단계를 보려면 ‘오늘 루틴 보기’를 선택하세요.” 안내 문구를 노출한다.
4. 루틴 진행 정보 카드
현재 진행 중인 루틴의 시작일과 진행 일차를 노출한다.
시작일은 “루틴 시작일 yyyy.mm.dd” 형식으로 표시한다.
진행 일차는 “시작한 지 n일” 형식으로 표시한다.
루틴 진행 정보는 조회 전용으로 표기한다.
우측에는 “루틴 종료하기” 버튼을 노출한다.
“루틴 종료하기” 버튼 클릭 시 루틴 종료 확인 화면으로 이동한다.
루틴 종료 시 완료 스킨노트 생성 플로우로 이어진다.
5. 이번주 변화과정기록 카드
화면 하단에 “이번주 변화과정기록” 카드를 노출한다.
카드에는 이미지 아이콘과 “이번 주 피부 변화를 사진으로 남겨보세요.” 안내 문구를 노출한다.
카드 클릭 시 변화 과정 기록 화면으로 이동한다.
이번 주 변화 기록을 이미 완료한 경우 문구를 “이번 주 변화 기록을 완료했어요.”로 변경한다.
6. 하단 안내 문구
화면 하단에 “매일의 기록이 피부 변화를 만들어요!” 안내 문구를 노출한다.
7. 하단 네비게이션 바
홈 클릭 시 홈 화면으로 이동한다.
케어로그 클릭 시 케어로그 화면으로 이동한다.
스킨서랍장 클릭 시 스킨서랍장 화면으로 이동한다.
마이페이지 클릭 시 마이페이지 화면으로 이동한다.
현재 화면은 케어로그이므로 케어로그 탭을 활성 상태로 표시한다.
현재 화면의 탭을 다시 클릭해도 별도 동작은 하지 않는다.
