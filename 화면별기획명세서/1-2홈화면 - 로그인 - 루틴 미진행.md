메인 메시지 문구 (와프 표기)
일러스트 영역: 이미지 추가(첨부) 

------------------------------------------------------------------------------

2. 내 피부 프로필 영역
버튼 형식
피부 타입: 미등록
주요 고민: 미등록
안내 문구: (와프 표기)

------------------------------------------------------------------------------

3. 오늘 루틴 등록하기 버튼 
버튼 클릭 → 2번 화면(피부 프로필 등록)으로 이동
피부 프로필 등록 완료 후 → 3번 화면(루틴 등록)으로 이동

------------------------------------------------------------------------------

4. 명예의 스킨노트 영역
[저장 많은 카드], [댓글 많은 카드], [좋아요 많은 카드] 등 각 반응 지표가 높은 스킨노트 노출
현재 사용자는 스킨노트 미공유 상태이므로 카드 상세 내용은 블러 처리  
더보기 클릭 → 7번(경험게시판)이동

------------------------------------------------------------------------------

5. 이번 주 참여 기록 빈 상태 영역
루틴 미등록 상태이므로 실제 주간 기록 데이터 없음
빈 상태 카드 문구: 와프 표기
말풍선 문구: 와프 표기 → 내일 이야기 해보기 
영역 클릭→ 해당 영역은 정보 확인용 영역으로 클릭 동작 없음

------------------------------------------------------------------------------

6. 하단 네비게이션 바
현재 선택된 탭 이모지 강조 색 (FF617D)
[홈], [케어 로그], [경험 게시판], [마이페이지] 구성
[홈] → 1번 화면(홈 화면) 이동
[케어로그] → 4번 화면(케어로그) 이동
[경험 게시판] → 7번 화면(경험 게시판) 이동
[마이페이지] → 8번 화면(마이페이지) 이동
