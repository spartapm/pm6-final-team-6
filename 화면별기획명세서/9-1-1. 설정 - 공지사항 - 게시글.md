9-1-1. 설정 - 공지사항 - 게시글

1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
클릭 시 이전 화면(9-1 공지사항 리스트)으로 이동한다.

2. 상단 타이틀 영역
화면 상단 중앙에 "공지사항" 텍스트를 고정 노출한다. (리스트 화면과 동일한 타이틀 유지)

3. 게시글 제목 영역
화면 상단에 선택한 공지글의 제목을 노출한다.
제목이 2줄 이상일 경우 줄바꿈되어 노출되며, 레이아웃이 깨지지 않는다.

4. 게시글 본문 영역
제목 하단에 공지글 본문 내용을 노출한다.
본문 내용은 텍스트 조회 전용이며, 클릭 시 별도 동작이 발생하지 않는다.
본문 내용이 화면 높이를 초과할 경우 세로 스크롤로 열람 가능하다.
이미지가 포함된 공지글의 경우 본문 내 이미지를 함께 노출한다.

5. 작성일 영역
화면 하단 우측에 "작성일: YYYY.MM.DD" 형식으로 공지글 작성일을 노출한다.
날짜는 조회 전용이며 클릭 시 별도 동작이 발생하지 않는다.

6. 하단 네비게이션 바
홈, 케어로그, 스킨서랍장, 마이페이지 탭을 노출한다.
마이페이지 탭이 활성 상태로 표시된다.
각 탭 클릭 시 해당 화면으로 이동한다.
