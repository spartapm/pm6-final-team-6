1. 상단 인사 / 메인 메시지 영역
메인 메시지 문구: 안녕하세요 오늘도 피부 기록 해볼까요?
일러스트 영역: 이미지 추가(1. 홈화면_1번 일러스트 영역 이미지)  
  
---------------------------------------------------------------
 
2. 내 피부 프로필 
선택된 타입및 고민: 번트 형식
피부타입: 건성, 지성, 복합성, 민감성
주요고민: 여드름/트러블, 색소침착, 모공/피지, 주름/탄력, 민감/붉음증,보습/건조함, 미백/칙칙함, 각질 

---------------------------------------------------------------

3. 오늘 루틴 기록하기 버튼
루틴이 등록되어 있는 경우: 4번화면(케어로그) 이동 

---------------------------------------------------------------

4. 명예의 스킨노트  영역
[저장 많은 카드], [댓글 많은 카드], [좋아요 많은 카드]각 반응 지표가 높은 경험 카드 노출
프로필 이미지 아래 닉네임 표시 
더보기 클릭 → 7번(경험게시판)이동
각 인기 게시물 중 대표 이미지: 카드 등록 사용자 프로필 이미지 노출 
                   
---------------------------------------------------------------

5. 이번 주 참여 기록 요약 영역 
1주 간 기록 상태 → 요일별 표시(순서:일 월 화 수 목 금 토)
각 요일 별 사용자 참여 여부 확인
참여: 제작 캐릭터 (웃음)
미참여:제작 캐릭터 (무표정)
기록없음: 빈 상태 (원형)
당일 요일: 원형에 들어있는 캐릭터 (참여도에 따라 달라짐) 

---------------------------------------------------------------

6. 하단 네비게이션 바
현재 선택된 탭 이모지 강조
[홈], [케어 로그], [경험 게시판], [마이페이지] 구성
[홈] → 1번(홈 화면)이동
[케어로그] → 4번 화면(케어로그)이동
[경험 게시판] → 7번 화면(경험 게시판) 이동
[마이페이지] → 8번 화면(마이페이지) 이동
