1. 뒤로가기 버튼
뒤로가기 버튼 클릭 시 로그인 화면 진입 직전 화면으로 이동한다.
마이페이지에서 로그인 화면으로 진입한 경우, 비로그인 마이페이지 화면으로 이동한다.
2. 로고 및 서비스 소개 영역
서비스 캐릭터, 서비스명 ANA, 서비스 설명 문구를 노출한다.
서비스 설명 문구는 “내 피부 고민에 맞는 루틴을 기록해요”로 표기한다.
해당 영역은 조회 전용이며 별도 클릭 동작은 제공하지 않는다.
3. 이메일 입력란
사용자의 이메일을 입력받는다.
이메일 입력란이 비어 있는 상태에서 로그인을 시도할 경우, 입력창 하단에 “이메일을 입력해주세요.” 안내 문구를 노출한다.
이메일 형식이 올바르지 않은 경우, 입력창 하단에 “올바른 이메일 형식으로 입력해주세요.” 안내 문구를 노출한다.
4. 비밀번호 입력란
사용자의 비밀번호를 입력받는다.
비밀번호 입력란이 비어 있는 상태에서 로그인을 시도할 경우, 입력창 하단에 “비밀번호를 입력해주세요.” 안내 문구를 노출한다.
비밀번호는 마스킹 처리하여 표시한다.
5. 자동 로그인 체크박스
기본값은 미체크 상태이다.
체크한 상태로 로그인에 성공하면 다음 접속부터 로그인 상태를 유지한다.
사용자가 로그아웃할 경우 자동 로그인 상태는 해제된다.
6. 비밀번호 찾기
“비밀번호 찾기” 클릭 시 비밀번호 찾기 화면으로 이동한다.
MVP에서 비밀번호 찾기 기능을 제공하지 않는 경우, “준비 중인 기능입니다.” 토스트 메시지를 노출한다.
7. 로그인 버튼
이메일과 비밀번호가 모두 입력된 경우 로그인 버튼을 활성화한다.
이메일 또는 비밀번호가 입력되지 않은 경우 로그인 버튼은 비활성화한다.
로그인 버튼 클릭 시 입력된 이메일과 비밀번호로 로그인을 시도한다.
로그인 성공 시 로그인 전 사용자가 머물던 화면으로 이동한다.
마이페이지에서 로그인한 경우 로그인 상태의 마이페이지 화면으로 이동한다.
로그인 실패 시 “이메일 또는 비밀번호가 일치하지 않습니다.” 안내 문구를 노출한다.
8. 회원가입 버튼
“회원가입” 클릭 시 회원가입 화면으로 이동한다.
회원가입 완료 후 로그인 상태로 전환하거나, 로그인 화면으로 이동하는 정책은 회원가입 플로우에서 정의한다.
