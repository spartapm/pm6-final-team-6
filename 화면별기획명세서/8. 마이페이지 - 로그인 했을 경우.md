설정 아이콘 영역• 클릭 시 설정 화면으로 이동(mvp 이후 v.2에 추가 예정)• 알림 설정, 로그아웃 메뉴 확인 가능• 설정 화면이 MVP 범위에 포함되지 않는 경우, 설정 아이콘은   숨기거나 “준비 중입니다” 토스트를 노출한다.

마이 프로필 영역• 사용자의 프로필 이미지, 닉네임을 노출한다.• 프로필 이미지가 없는 경우 기본 캐릭터 이미지를 노출한다.• 프로필 이미지 클릭 시 이미지 변경 기능을 제공한다.• 이미지 변경 방식은 앨범에서 선택으로 제공한다.• 업로드 가능한 파일 형식은 jpg, png이며, 최대 용량은 10MB로 제한한다.• 파일 형식 또는 용량 조건을 벗어날 경우 오류 안내 문구를 노출한다.

-------------------------------------------------------------------
진행중인 루틴 영역• 현재 진행 중인 루틴이 있는 경우 루틴 카드를 노출한다.• 루틴 카드에는 피부 고민명, 진행 일차, 사용 제품 요약 정보를 표시한다.• 루틴 카드 클릭 시 케어로그 또는 변화 과정 기록 화면으로 이동한다.• 진행 중인 루틴이 없는 경우 “진행 중인 루틴이 없어요” 빈 상태 화면과 루틴 등록 CTA를 노출한다.

스킨노트 모아보기 – 더보기 버튼• 사용자가 완료한 스킨노트를 최대 3개까지 가로 카드 형태로 노출한다.• 스킨노트 카드는 종료된 루틴에서 생성된 스킨노트만 노출한다.• 각 카드에는 완료 배지, 대표 캐릭터 또는 대표 이미지, 피부 고민명, 사용 기간을 표시한다.• 스킨노트 카드 클릭 시 해당 스킨노트 상세 화면으로 이동한다.•  등록된 스킨노트가 없는 경우 “아직 완성된 스킨노트가 없어요” 빈 상태를 노출한다.(mvp 이후 v.2 추가 예정)
-------------------------------------------------------------------

스킨노트 모아보기 영역• 더보기 버튼 클릭 시 내가 완료한 스킨노트 전체 목록 화면으로 이동한다.• 해당 기능이 MVP 이후 범위라면 더보기 버튼은 노출하지 않는다.
-------------------------------------------------------------------

완료 배지 영역• 완료 배지는 종료된 루틴을 기반으로 생성된 스킨노트에 표시한다.• 중도 포기한 루틴은 완료 배지를 표시하지 않는다.• 완료 배지 클릭 시 별도 동작은 제공하지 않는다.
-------------------------------------------------------------------
나의 관심• MVP에서는 나의 관심 영역을 노출하지 않는다.• v2에서 최근 본 스킨노트, 저장한 스킨노트, 좋아요한 스킨노트, 신고 내역을 추가한다.
