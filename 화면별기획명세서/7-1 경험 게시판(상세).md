7-1. 스킨노트 상세 화면
1. 상단 헤더 영역
화면 상단에 “스킨노트 상세” 타이틀을 노출한다.
좌측에는 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 이전 화면인 스킨서랍장 목록 화면으로 이동한다.
2. 안내 배너 영역
스킨노트 상세 화면 상단에 안내 배너를 노출한다.
안내 문구는 “경험카드만 공유할 수 있어요. 게시글에는 추가 글을 작성할 수 없어요.”로 표시한다.
우측 X 버튼 클릭 시 배너를 닫는다.
배너를 닫은 사용자는 이후 동일 배너를 다시 노출하지 않는다.
안내 배너 노출 여부는 사용자 단위로 저장한다.
3. 스킨노트 카드 작성자 정보 영역
스킨노트 카드 상단에 작성자 프로필 정보를 노출한다.
작성자 프로필 영역에는 프로필 이미지, 닉네임, 피부 타입, 나이대, 민감도, 작성 시간을 표시한다.
프로필 이미지가 없는 경우 기본 프로필 이미지를 노출한다.
작성 시간은 “3시간 전”과 같은 상대 시간 형식으로 표시한다.(24시간이 지나면, 1일 전, 1달 전, 1년 전 이런 식으로 표기) 
우측 더보기 아이콘 클릭 시 액션 시트를 노출한다.
타인 작성 스킨노트의 경우 액션 시트에는 “신고하기”, “숨기기”를 노출한다.
본인 작성 스킨노트의 경우 액션 시트에는 “삭제하기”를 노출한다.
4. 스킨노트 기본 정보 영역
스킨노트 카드 본문에 “경험 카드” 라벨을 노출한다.
제목은 “14일 사용 후기” 형식으로 표시한다.
변화 태그는 해시태그 칩 형태로 노출한다.
태그 예시는 “#붉은기 완화”, “#피부결 개선”, “#수분 충전”으로 표시한다.
경험카드의 제목과 태그는 조회 전용으로 제공한다.
5. 사용 제품 영역
사용자가 루틴에 등록한 제품 목록을 루틴 순서대로 노출한다.
각 제품은 제품 이미지와 제품명을 함께 표시한다.
제품 이미지는 최대 4개까지 우선 노출한다.(가로 스크롤로 제공)
제품 이미지가 없는 경우 기본 이미지 플레이스홀더를 노출한다.
MVP에서 제품 상세 화면을 제공하지 않는 경우 제품 클릭 시 별도 동작은 제공하지 않는다.
6. 변화 과정 영역
사용자가 기록한 변화 과정을 시간 순서대로 노출한다.
변화 과정은 사용 전, 7일차, 10일차, 14일차 등 기록 일자 기준으로 표시한다.
사진이 등록된 기록은 썸네일 이미지를 노출한다.
사진이 없는 기록은 사용자가 선택한 피부 컨디션 이모지와 변화 여부 텍스트를 노출한다.
사진 썸네일 클릭 시 이미지 크게 보기 팝업을 노출한다.
7. 변화 태그 영역
사용자가 선택한 변화 태그를 칩 형태로 노출한다.
변화 태그는 스킨노트 생성 시 저장된 값을 그대로 표시한다.
태그 예시는 “#붉은기 완화”, “#수분 개선”, “#피부결 정돈”으로 표시(가로스크롤로 제공)한다.
변화 태그는 조회 전용으로 제공한다.
8. 사용 기간 / 난이도 / 체감 변화 영역
스킨노트 하단에 사용 기간, 루틴 난이도, 체감 변화를 노출한다.
사용 기간은 “14일” 형식으로 표시한다.
루틴 난이도는 “쉬웠어요”, “보통이에요”, “어려웠어요” 중 저장된 값을 표시한다.
체감 변화는 별점과 숫자를 함께 표시한다.
예시는 “★★★★☆ 4”로 표시한다.
해당 정보는 조회 전용으로 제공한다.
9. 카드 반응 영역
스킨노트 하단에 저장, 도움돼요, 댓글 수를 노출한다.
저장 클릭 시 저장/저장 취소 상태를 토글한다.
저장 상태가 변경되면 저장 수를 즉시 업데이트한다. 
도움돼요 상태가 변경되면 도움돼요 수를 즉시 업데이트한다.
댓글 클릭 시 댓글 입력 영역으로 포커스한다.
비로그인 사용자가 저장 또는 도움돼요를 클릭할 경우 로그인 안내 모달을 노출한다.
10. 댓글 목록 영역
스킨노트 하단에 댓글 목록을 노출한다.
댓글 수는 “댓글 32” 형식으로 표시한다.
기본 정렬은 최신순으로 제공한다.
정렬 버튼 클릭 시 최신순 / 좋아요순 옵션을 제공한다.
각 댓글에는 작성자 프로필 이미지, 닉네임, 댓글 내용, 작성 시간, 좋아요 수를 노출한다.  
댓글 우측 더보기 아이콘 클릭 시 액션 시트를 노출한다.
본인 댓글의 경우 “삭제하기”를 노출한다.
타인 댓글의 경우 “신고하기”를 노출한다.
v.1에서는 대댓글 기능은 제공하지 않는다.
11. 댓글 입력 영역
화면 하단에 댓글 입력창을 고정 또는 하단 영역에 노출한다.
입력창 placeholder는 “따뜻한 댓글을 남겨주세요.”로 표시한다.
댓글 입력 전에는 등록 버튼을 비활성화한다.
댓글을 1자 이상 입력하면 등록 버튼을 활성화한다.
댓글 등록 클릭 시 댓글을 저장하고 댓글 목록에 즉시 반영한다.
댓글 입력창에는 텍스트 입력을 기본으로 제공한다.
이미지 첨부와 이모지 기능은 MVP 이후 기능으로 분리할 수 있다.
비로그인 사용자가 댓글 입력창을 클릭할 경우 로그인 안내 모달을 노출한다.
