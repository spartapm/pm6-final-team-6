1. 안내 배너 ("스킨노트만 공유할 수 있어요")
- 최초 진입(첫 방문) 유저에게만 노출한다.
- (X) 아이콘 클릭 시 배너를 닫음 처리하며, 이후 재접속 시에도 다시 노출하지 않는다(서버 또는 유저 단위로 노출 여부 저장).
-------------------------------------------------------------------------------
2. 탭 메뉴 (전체 / 저장 / 인기)
- "전체" 클릭 시 전체 스킨노트 피드를 노출한다(기본 선택 탭).
- "저장" 클릭 시 내가 저장한 스킨 노트만 필터링하여 노출
- "인기" 클릭 시 도움돼요를 기준으로 인기순 스킨 노트를 노출
-------------------------------------------------------------------------------
3. 필터 칩 영역 (조건 선택 / 피부타입 / 피부고민 / 사용 기간)
사진 필터는 터치식 토글로 제공
 기본값 : OFF 상태
 토글 ON → 사진이 포함된 게시글만 노출
 토글 OFF → 사진 포함 여부와 관계없이 전체 게시글을 노출
- 복수 선택 시
 피부고민, 피부타입, 기간 순서로 필터 표시 (예 : 건성/민감성 / 붉은기 /14일)
- 필터 칩 개수가 화면 폭을 넘는 경우 가로 스크롤 표시
-------------------------------------------------------------------------------
4. 조건 선택 필터 팝업 (조건 선택 클릭 시 노출)
- 닫기(X) 클릭 시 팝업을 닫으며, 선택 변경사항은 저장하지 않고 이전 적용 상태를 유지.
- 피부 타입 기본값 : 전체
- 피부 타입 : 전체 / 건성 / 지성 /복합성 / 민감성 
- 피부고민 기본값 : 전체
- 피부 고민 : 전체 / 여드름・트러블 / 붉은기 / 모공・피지 / 속건조 / 탄력 저하
- 사용 기간 기본값 : 전체
- 사용 기간 : 7일 미만 / 7일 이상 / 14일 이상
- "초기화" 버튼 클릭 → 모든 조건을 기본값("전체")으로 리셋, 팝업은 닫히지 않고 팝업 내에서만 초기화
- "적용하기" 버튼
클릭 시 선택한 조건을 게시글 목록에 반영한다.
조건 적용 후 바텀시트를 닫는다.
선택된 조건이 있을 경우 상단 필터 칩에 선택 상태를 반영한다.
조건에 맞는 카드가 없는 경우 빈 상태 화면을 노출한다.
-------------------------------------------------------------------------------
5. 스킨노트 미리보기 - 카드 형태로 노출
- 작성자 프로필 사진, 닉네임, 피부 타입・피부고민・나이대 노출
- 작성 시간 : 공유 시간 기준
- ’ ⋮ ‘ 아이콘 
본인 스킨노트 일 경우에만 노출
클릭 시 : 삭제하기 노출 
 [게시글 본문]
제목 : (사용기간) 사용 후기
변화태그 칩 형태로 노출 (최대 5개까지 노출) 가로 영역을 초과할 경우 줄바꿈 하여 표기
사용 기간 / 체감 변화(별점) 표시
- 사용 기간 : 입력한 총 사용일수 표기
- 체감 변화 : 6번(스킨노트완성)화면 체감변화 데이터 표기
[하단 반응 정보 영역]
저장, 도움돼요, 댓글 영역을 클릭해도 별도 동작이 발생하지 않음.
- 스킨노트 전체 영역 클릭 시
로그인 유저 → 해당 스킨노트 상세 화면(7-1)으로 이동
비로그인 유저 → 8-1(마이페이지 로그인) 화면으로 이동
-------------------------------------------------------------------------------

6. 하단 네비게이션 바
- 공통 컴포넌트를 참고한다(홈 화면 정의와 동일).
