8-3-3. 비밀번호 찾기 - 새 비밀번호 설정
1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 비밀번호 찾기 2단계 화면으로 이동한다.
2. 상단 타이틀 영역
화면 상단 중앙에 “비밀번호 찾기” 텍스트를 노출한다.
타이틀 하단에는 “3 / 3 · 새 비밀번호를 설정해주세요” 안내 문구를 노출한다.
3. 새 비밀번호 입력란
새 비밀번호 입력란을 노출한다.
입력란 placeholder는 “새 비밀번호 입력 (8자 이상, 영문+숫자)”로 표시한다.
새 비밀번호는 필수 입력 항목이다.
비밀번호는 8자 이상, 영문과 숫자를 포함해야 한다.
비밀번호는 마스킹 처리하여 표시한다.
조건을 충족하지 않을 경우 입력창 하단에 “비밀번호는 8자 이상, 영문과 숫자를 포함해야 합니다.” 안내 문구를 노출한다.
4. 새 비밀번호 확인 입력란
새 비밀번호 확인 입력란을 노출한다.
입력란 placeholder는 “새 비밀번호 확인”으로 표시한다.
새 비밀번호 확인은 필수 입력 항목이다.
비밀번호 확인 값은 마스킹 처리하여 표시한다.
새 비밀번호와 일치하지 않을 경우 입력창 하단에 “비밀번호가 일치하지 않습니다.” 안내 문구를 노출한다.
5. 변경완료 버튼
화면에 “변경완료” 버튼을 노출한다.
새 비밀번호와 새 비밀번호 확인이 모두 입력되고, 비밀번호 조건을 충족하며, 두 값이 일치할 경우 버튼을 활성화한다.
필수 입력값이 비어 있거나 조건을 충족하지 않는 경우 버튼은 비활성화 상태로 표시한다.
변경완료 버튼 클릭 시 비밀번호 변경을 요청한다.
비밀번호 변경 성공 시 “비밀번호가 변경되었습니다. 새 비밀번호로 로그인해주세요.” 안내 문구를 노출한 뒤 로그인 화면으로 이동한다.
비밀번호 변경 실패 시 “비밀번호 변경에 실패했습니다. 다시 시도해주세요.” 안내 문구를 노출한다.
6. 하단 진행 안내 문구
화면 하단에 “화면 3/3 — 새 비밀번호 설정” 문구를 노출한다.
