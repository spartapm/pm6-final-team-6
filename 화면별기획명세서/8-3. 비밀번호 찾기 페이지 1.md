비밀번호 찾기는 이메일 인증을 통해 새 비밀번호를 설정하는 3단계 플로우로 구성한다.
8-3-1. 비밀번호 찾기 - 이메일 인증 요청
1. 뒤로가기 버튼
화면 좌측 상단에 뒤로가기 아이콘을 노출한다.
뒤로가기 아이콘 클릭 시 로그인 화면으로 이동한다.
2. 상단 타이틀 영역
화면 상단 중앙에 “비밀번호 찾기” 텍스트를 노출한다.
타이틀 하단에는 “1 / 3 · 가입한 이메일을 입력해주세요” 안내 문구를 노출한다.
3. 이메일 입력란
이메일 입력란을 노출한다.
입력란 placeholder는 “이메일 입력”으로 표시한다.
이메일은 필수 입력 항목이다.
이메일 형식이 올바른 경우 “인증번호 받기” 버튼을 활성화한다.
이메일 미입력 상태에서 인증번호 받기를 시도할 경우 입력창 하단에 “이메일을 입력해주세요.” 안내 문구를 노출한다.
이메일 형식이 올바르지 않은 경우 입력창 하단에 “올바른 이메일 형식으로 입력해주세요.” 안내 문구를 노출한다.
가입되지 않은 이메일인 경우 입력창 하단에 “가입되지 않은 이메일입니다.” 안내 문구를 노출한다.
4. 인증번호 받기 버튼
화면에 “인증번호 받기” 버튼을 노출한다.
이메일 입력값이 비어 있거나 이메일 형식이 올바르지 않은 경우 버튼은 비활성화 상태로 표시한다.
버튼 클릭 시 입력한 이메일로 인증번호 발송을 요청한다.
인증번호 발송 성공 시 비밀번호 찾기 2단계 화면으로 이동한다.
인증번호 발송 실패 시 실패 원인에 맞는 안내 문구를 노출한다.
5. 하단 진행 안내 문구
화면 하단에 “화면 1/3 — 이메일 인증 요청” 문구를 노출한다.
